# Security Policy

This is a study repository — algorithm drills, printable reference material,
and practice tooling. It ships no runtime service, collects no data, and the
site is static.

If you believe you've found a security issue (for example in the devcontainer
bootstrap, CI workflows, or a script), please report it privately via
[GitHub Security Advisories](../../security/advisories/new) rather than a
public issue. Reports are read by the maintainer; expect an initial response
within a week.
