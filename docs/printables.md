# Printables

Use the booklet for a complete offline reference. Print individual sheets when
you want one compact prompt beside the editor or board.

## Full booklet

The booklet combines the pattern guides, algorithm explanations and code, and
the interview-practice method. Its page count changes with the source. Build a
fresh copy before printing.

## Build locally

```bash
just packet
just pdf-all
```

`just packet` builds the booklet. `just pdf-all` builds all reference-sheet
PDFs. For paper, duplex printing with a long-edge binding keeps the stack
compact. Grayscale is fine.
