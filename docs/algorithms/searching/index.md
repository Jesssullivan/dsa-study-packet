# Searching

| Problem | Complexity | Key Pattern |
|---------|-----------|-------------|
| [Binary Search](binary_search.md) | `O(log n)` | Sorted array lookup or any monotonic predicate search — "find target", |
| [Find Minimum Rotated](find_minimum_rotated.md) | `O(log n)` | Pivot finding in a rotated sorted array — "find rotation point", |
| [Search Rotated Array](search_rotated_array.md) | `O(log n)` | Invariant-based binary search — array is sorted but shifted/rotated. |
